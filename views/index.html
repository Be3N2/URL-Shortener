Url Shortener Microservice

How to Use

Can shorten a url by adding the extension /shorten/yourUrlHere and will get
back a json object with the new shortened url, that will redirect to the inputted url.
It creates a 3 digit hexadecimal code for each inputted url.

A few example calls:

  • https://url-shortener-node.glitch.me/shorten/www.google.com
  • https://url-shortener-node.glitch.me/shorten/https://about.freecodecamp.org/

The response in json format:

  • {"originalUrl": "www.google.com", "newUrl": "http://url-shortener-node.glitch.me/c23"}
  • {"originalUrl": "https://about.freecodecamp.org", "newUrl": "http://url-shortener-node.glitch.me/bf4"}

The newUrl value will be a redirect to inputted site when followed:

  • http://url-shortener-node.glitch.me/c23 Will link to Google.
  • http://url-shortener-node.glitch.me/bf4 Will link to Freecodecamp.

Project made by Ben Wakefield for Freecodecamp. Source code and more info available on Github.
